Atributos:

title:

Este atributo permite que se muestre el nombre de una etiqueta o elemento, cuando pasamos por encima de ella con el cursor.

style:

El atributo style, el cuel nos permite agregar comandos css, por ejemplo usando style = 'color:blue' obtenemos este parrafo. usando font-size = cantidadpx definimos el tamaño de la letra usando font-family = "tipo de letra" definimos la fiente de la letra
Si usamos el atributo style en el body, junto a "background-color:ColorFondo" podemos definir un color de fondo para la pagina.
Podemos agregar otros atributos de estilos separandolos por ;
Todos estos colores pueden cargarse mediante un codigo rgb, un código exadecimal o incluso un color de los ya establecidos en HTML
